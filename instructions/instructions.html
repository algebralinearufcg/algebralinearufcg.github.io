Como usar os notebooks¶

Os arquivos com extensão .ipynb são chamados de notebooks. Um notebook é composto por células, essas que obedecem uma das normas vigentes de boa prática de programação: a ideia da programação sequencial. Uma célula pode ser de dois tipos: célula de código (Code) ou célula de texto (Markdown). As células de texto são independentes entre si em um viés computacional, pois a execução da mesma não vai interferir nas células de código. Sobre o viés do aluno, é interessante para o seu entendimento que a leitura das mesmas seja feita sequencialmente, dado que algumas células desenvolvidas anteriormente ajudam no entendimentos das desenvolvidas posteriormente (sequencial). Desse modo, é fundamental que:

TODAS AS CÉLULAS DE CÓDIGO DEVEM SER EXECUTADAS, DO INÍCIO AO FIM, DADO QUE AS CÉLULAS POSTERIORES SÃO DEPENDENTES DAS CÉLULAS ANTERIORES

Caso contrário, erros de interpretação surgirão, interrompendo o funcionamento do programa.

Abrindo um notebook¶

Para quem optou por instalar na sua máquina o ambiente de trabalho do Python (você fez uma excelente escolha!), pode seguir as seguintes instruções para abrir um notebook em sua máquina:

  1. Abra o prompt de comando.
  2. Digite jupyter notebook, caso tenha optado pelo método de instalação pip install jupyter. Digite jupyter lab, caso tenha optado pela opção de instalação pip install jupyter lab (a segunda versão é mais recente e com mais ferramentas).

Jupyter Notebook tem o mesmo funcionamento do Jupyter Lab, a única diferença é que o segundo é uma versão do aplicativo mais recente que a primeira.

  1. Após ter feito isso, será criado no seu prompt de comando uma espécie de servidor local e o aplicativo será aberto em seu browser padrão.

NÃO FECHE O SEU PROMPT DE COMANDO ENQUANTO ESTIVER USANDO O JUPYTER NOTEBOOK OU JUPYTER LAB, CASO CONTRÁRIO, O KERNEL DO APLICATIVO SERÁ ENCERRADO JUNTO.

  1. Caso você queira criar um novo notebook, clique na opção "New Launcher". Dentro do programa:
    • 4.1 Ícone do disquete("Save the notebook contents and create checkpoint"): salva o seu programa;
    • 4.2 Ícone de +("Insert a cell below"): Insere uma célula no seu notebook;
    • 4.3 Ícone de tesoura("Cut the selected cells"): corta alguma célula selecionada do seu programa;
    • 4.4 Ícone de copiar("Copy the selected cells"): copia uma célula do seu programa;
    • 4.5 Ícone da prancheta("Paste cells from the clipboard"): cola a célula copiada;
    • 4.6 Ícone de dar play("Run the selected cells and advance"): interpreta a célula selecionada;
    • 4.7 Ícone de parar("Interrupt the Kernel"): Interrompe a interpretação da célula em processo;
    • 4.8 Ícone de reiniciar("Restart the Kernel"): Reinicia a interpretação da célula em processo;
    • 4.9 Opção Markdown: Transforma a célula selecionada em uma célula de texto;
    • 4.10 Opção Code: Transforma a célula selecionada em uma célula de código;
    1. Caso você queira criar uma pasta, clique na opção "New folder".
    2. Caso você queira adicionar um novo arquivo .ipynb do seu computador, clique na opção "Upload Files".

Para quem optou usar o Google Colab Notebook, pode seguir as seguintes instruções para executar um notebook em um servidor pelo Google Browser.

  1. Logo no início, aparecerá uma janela com diversas opções, nela você pode fazer os seguintes passos:
    • 1.1 Na aba "Examples", temos uma série de programas .ipynb fornecidos pelo Google para familiarizar o usuário com o aplicativo;
    • 1.2 Na aba "Recent", é possível ver os programas .ipynb acessados recentemente;
    • 1.3 Na aba "Google Drive", você verá todos os seus programas criados;
    • 1.4 Na aba "GitHub", pode-se copiar a URL do repositório que tem uma extensão .ipynb e usá-lo para abrir o programa do GitHub no Google Colab Notebook de forma direta;
    • 1.5 Na aba "Upload" você pode adicionar um programa .ipynb do seu computador diretamente no Google Colab Notebook.

Todos os programas criados no Google Colab Notebook são salvos no Google Drive e vinculado a conta do usuário.

PARA USUÁRIOS DO GOOGLE COLAB NOTEBOOK, ESSA SERÁ A FORMA DE ACESSAR OS PROGRAMAS DO NOSSO REPOSITÓRIO

  1. Na opção "New Python3 Notebook" você conseguirá criar um novo notebook. Vamos agora aprender os comandos para prosseguir com essa criação.
    • 2.1 Na opção "+ Code", você pode adicionar uma célula de código;
    • 2.2 Na opção "+ Text", você pode adicionar uma célula de texto;
    • 2.3 Clicando em "file": salva o seu programa;
    • 2.4 Ícone de lixeira: exclui alguma célula selecionada do seu programa;
    • 2.5 Nos três pontos para ter acesso a mais opções na célula, você pode achar a opção "Copy cell", para copiar a célula;
    • 2.6 Ao lado de cada célula, há uma opção para interpretar ela, assim, você pode executar as células de código;
    • 2.7 Clicando na opção "runtime", você pode achar a opção "Run all", para interpretar todas as células do programa de uma vez só;

Baixando Arquivos do GitHub¶

Baixar pelo próprio GitHub¶

  1. Primeiramente, clique no arquivo que você deseja baixar no GitHub.
  2. Após clicar no arquivo desejado com extensão .ipynb, você conseguirá ver o arquivo, mas não poderá interagir com ele.
  3. Clique na opção Raw e, após isso, clique em "Ctrl + S" para salvar o arquivo em seu computador.
  4. Repare que o tipo do arquivo está como um documento de texto, você deve modificar essa opção para "Todo tipo de arquivo" e modificar também o nome do documento. A extensão .ipynb deve ser mantida.
  5. O nome do documento deve terminar com .ipynb, respeitando as regras ao colocar o nome no arquivo, para que ele seja interpretado.
  6. Abra o seu Jupyter Notebook ou Google Colab Notebook.
  7. Faça o upload do arquivo e pronto, você terá um arquivo .ipynb na qual você pode interagir, editar, remover ou adicionar células.

Usando o nbviewer¶

O próprio Jupyter Project fornece uma ferramenta para que o usuário consiga baixar e visualizar arquivos .ipynb a partir do GitHub. Segue o link do site para acessar o nbviewer.

  • nbviewer
  1. Insira o link do programa disponível do repositório e clique em pesquisar.
  2. Clique em "Download Notebook" para baixar o arquivo na sua máquina.

Usando o binder para compilar os programas¶

Uma opção terciária para executar os programas .ipynb é usando o binder, do próprio nbviewer. Veja os passos abaixo:

  1. Insira o link do programa disponível no repositório e clique em pesquisar.
  2. Clique na opção "Execute on Binder".
  3. Abrirá no browser um aplicativo Jupyter Notebook.
